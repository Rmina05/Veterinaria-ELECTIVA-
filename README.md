# Proyecto-electiva-

🐾 Sistema Web de Gestión Veterinaria
es una aplicación web desarrollada en PHP, diseñada para optimizar la gestión de citas, mascotas, historias clínicas, recetas y facturación en una clínica veterinaria.
Permite a administradores, veterinarios y recepcionistas interactuar en un entorno digital ágil y seguro.



🚀 Características Principales

👩‍💼 Gestión de usuarios y roles: administrador, veterinario y recepcionista 

🐶 Registro de mascotas y clientes con historial médico.

📅 Agenda de citas veterinarias con filtro por fecha, estado y profesional.

🧾 Facturación y generación de PDF de facturas.

💊 Recetas médicas generadas desde la historia clínica.

📅 control de stock y servicios 

🩺 Módulo de historias clínicas con diagnóstico y tratamiento.

📈 Panel administrativo con indicadores de gestión.

🔐 Inicio de sesión y registro de usuarios con autenticación por rol.








🛠️ Tecnologías Utilizadas

PHP (backend)

HTM + CSS (frontend)

MySQL (Base de datos)

XAMPP (Servidor local)

Git y GitHub (Control de versiones)
